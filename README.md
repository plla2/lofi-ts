## 💻 Repo소개
3가지의 lofi음악 모드와 여러가지 다양한 백그라운드 노이즈들을 선택하고, 볼륨을 조절함과 동시에 집중모드 타이머 설정을 통해 집중하는 시간을 정하고, 투두리스트를 통해 할일을 정리할 수 있는 웹앱

</br>
